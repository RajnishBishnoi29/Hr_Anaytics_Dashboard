Interactive HR Analytics Dashboard built with Power BI to analyze employee attrition, workforce demographics, job roles, and key HR metrics using DAX, Power Query, and data visualization techniques.
# HR Analytics Dashboard | Power BI

## Project Overview

This project is an interactive HR Analytics Dashboard developed in Power BI to help organizations analyze workforce data and understand employee attrition, demographics, and performance trends. The dashboard converts raw HR data into meaningful insights that support data-driven HR decisions.



## Objectives

* Analyze employee attrition trends.
* Monitor key HR metrics through interactive KPIs.
* Understand workforce demographics.
* Identify factors affecting employee retention.
* Support HR teams with actionable business insights.


## Tools & Technologies

* Power BI
* Power Query
* DAX
* Microsoft Excel



## Dashboard Features

* Executive KPI Cards
* Employee Attrition Analysis
* Department-wise Analysis
* Gender Distribution
* Age Group Analysis
* Job Role Insights
* Interactive Filters & Slicers
* Dynamic Visualizations



##  Key Insights

* Track overall employee count and attrition.
* Compare attrition across departments and job roles.
* Analyze workforce distribution by age and gender.
* Identify trends that help improve employee retention.
* Enable HR managers to make informed decisions.
